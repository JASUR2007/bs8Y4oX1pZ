# StoryVault — Тестовое задание

## Описание проекта
StoryVault — это мини-проект для публикации сообщений пользователями с возможностью 
прикрепления изображений, редактирования и удаления своих сообщений.
Проект выполнен на Yii2 с использованием Bootstrap 5 для фронтенда.
